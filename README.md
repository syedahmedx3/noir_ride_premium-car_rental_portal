# 🏎️ NOIR RIDE | Premium Car Rental <span style="float:right;">[![LinkedIn](https://img.shields.io/badge/LinkedIn-0077B5?style=flat&logo=linkedin&logoColor=white)](https://www.linkedin.com/in/syedahmedhasanx3/) [![Portfolio](https://img.shields.io/badge/Portfolio-Live-00df9a)](https://syedahmedx3.github.io/Portfolio/)</span> 
A high-performance, elite booking frontend engineered for luxury car rentals. NOIR RIDE blends a cinematic user experience with enterprise-grade logic, featuring real-time pricing, digital signatures, and a modular architecture ready for Odoo backend integration.

## ✨ Project Overview

This portal is designed for the premium GCC market (KSA Ready). It moves beyond a simple brochure by implementing a "Headless" philosophy—separating complex frontend state management (fleet filtering, currency conversion, and dynamic pricing) from the eventual backend API.

## 🚀 Key Features

* **⚡ Dynamic Pricing Engine:** Real-time rental calculation based on vehicle class, insurance tiers, seasonal demand, and premium add-ons.
* **🌍 Globalization (RTL/LTR):** Seamless Arabic/English localization with full Right-to-Left (RTL) UI support.
* **✍️ Digital Signature Pad:** Integrated HTML5 Canvas signature module for digital agreement acceptance.
* **🚗 Real-Time Fleet Grid:** Advanced filtering system for smooth category transitions (Luxury, Sports, SUVs, Electric).
* **📱 Immersive UX:** Cinematic animations powered by GSAP and ScrollTrigger, plus a custom magnetic cursor for a premium feel.
* **📁 Document Management:** Client-side verification zone for uploading licenses and IDs.

## 🧰 Technologies Used

* **HTML5 & CSS3:** Semantic structure with a focus on high-performance rendering.
* **JavaScript (ES6+):** Functional, state-driven logic handling all calculations and UI synchronization.
* **GSAP & ScrollTrigger:** High-end motion design and reveal animations.
* **Swiper.js:** Touch-optimized, high-performance car galleries.
* **I18n Engine:** Custom data-attribute driven localization system.

## 🛠️ Installation & Setup

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/syedahmedx3/noir_ride_premium-car_rental_portal.git
    ```

2.  **Navigate to the project directory:**

    ```bash
     cd noir_ride_premium-car_rental_portal
    ```
3.  **Open the project:**
* Simply open index.html in your browser or use the Live Server extension in VS Code.

## 🏗️ Backend Integration
This frontend is built with API-Sync Ready architecture. The bookingForm and rentalData objects are structured to facilitate easy binding with Odoo (XML-RPC/JSON-RPC) or a NestJS microservice for real-time fleet maintenance and order synchronization.
